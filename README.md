# So_long (Linux)

## Installation

Use make to the mandatory part.

```bash
make
```
Use make bonus to the bonus part.

```bash
make bonus
```
![Alt Text](https://s10.gifyu.com/images/compil.gif)

## Run

Choose corresponding map to the map folder.

```bash
./so_long_bonus map/map_valid_bonus.ber
```

## Preview

Use w, a, s, d to move.
